## How to run
B1. Clone repo:
```bash
git clone https://github.com/cloth-nest/cloth-nest-be.git
```
B2. Inside your terminal, go to the repo and run this command to get dependencies:
```bash
yarn install
```
B3: To run backend, in your terminal run this:
```bash
yarn start:dev
```
