## How to run
B1. Clone this repo:
```bash
git clone https://github.com/cloth-nest/cloth-nest-be.git
```

B2. Open terminal and go to the repo, then run:
```bash
yarn install
```

B3. To run backend, run this in terminal:
```bash
yarn start:dev
```
